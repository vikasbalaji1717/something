# Music Folder

Place a romantic MP3 file named **music.mp3** here.

Example sources (free, royalty-free):
- https://pixabay.com/music/search/romantic/
- https://freemusicarchive.org/genre/Ambient/

The music toggle button (🎵) in the top-right corner will play/pause this file.
It also auto-plays when the user clicks "Yes" 💖
